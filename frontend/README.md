VelevateU is a student empowerment and community-driven platform focused on providing guidance, volunteering opportunities, and growth-oriented resources to help students build confidence, skills, and a brighter future.

How can I edit this code?

There are several ways to edit and update this application.

Use the Deployment Platform

If the project is connected to a deployment platform (such as Render), you can open the deployed project dashboard and make changes directly.

Changes made through the platform may be automatically committed to this repository.

Use your preferred IDE

If you want to work locally using your own IDE, you can clone this repository and push changes. Any pushed updates will also be reflected in the deployed project.

The only requirement is having Node.js and npm installed.
Install using nvm: https://github.com/nvm-sh/nvm#installing-and-updating

Follow these steps:

# Step 1: Clone the repository
git clone https://github.com/velevateunavodaya/elevate-frontend.git

# Step 2: Navigate to the project directory
cd elevate-frontend

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server with auto-reloading and instant preview
npm run dev
